Go Menu
 for quick return to previously-displayed files (but this does not cause the OT Program window to move to the associated Observation - use this only for quickly having a look at a previous field. You will notice that the image window informs you that "base position is out of view".

Catalog

Mode and View
       
 Mode Section

Drag  - when selected, dragging the base position indicator in the image window causes the target position in the OT target component to update. The other two buttons in this section can be ignored.

GUIDE   - when selected, clicking on a star in the image window will select a guide star. The other two buttons in this section can be ignored.

View Section

Base  - causes the base position defined in the OT target component to be displayed as a small cross within a circle.

Guide etc.   - causes the guide position to be displayed. Note that you will need to have this clicked in order to select a guide star.

Sci Area  - causes the science area specified by the instrument component to be displayed.

Catalog - causes objects in the star catalogue selected in the "Catalogs" menu to be displayed.

Acq Cam - causes the field of the TSS's acquisition camera to be displayed. Not normally a useful function since most acquisition proceeds without using this camera.

Xhead - causes the range of motion of the crosshead (on which the guider is mounted) to be displayed.

Dichroic - causes areas of the field affected by the edges of the dichroic to be displayed.


Main image window
    

 Image Window Contents

[Purple] - Crosshead limits  - Approximate; be conservative when choosing guide stars near the edges of this circle. Note that UKIRT can guide down to 18th Magnitude (sometimes fainter in good seeng and dark of moon); as a rule of thumb, a 14th-magnitude star is the last to display visible signs of diffraction spikes. The radius of the circle is about 120 arcseconds.

[Green] - Dichroic edges - Again approximate; stay away from the green lines.

[Green] - base position - In this example, hard to see down in the centre.

[Green] - Guide star position.

[Cyan] - science field - in this case, the 90-arcsecond Michelle slit.


Pan and Zoom
    

 Pan and Zoom Window Contents

[Top] - Pan window  - when zoomed in, moving the yellow box causes the main window to pan around the image.

[Bottom] - Zoom window - A small subsection of the image, centred on the cursor position. Use for finessing guide star selection.
