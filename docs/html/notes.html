Notes regarding the adoption of the ot-0.5 Swing GUI (Gemini, jsky)  for the ATC version of the OT

  • In August 2000 I came up with an OT version which was based on ot-0.3 plus the ATC's instrument and telescope specific classes (mainly the package ot_ukirt)
  • The new ATC ORAC OT should be based on the ATC's current ORAC OT minus freebongo plus the swing based GUI of the ot-0.5 It turned out to be necessary to use rather many of the jsky ot-0.5 (See table below)
  • While I was working with version ot-0.5 a later version, ot-0.6, was releaded. I did not adopt ot-0.6 for the time being because it adopted a new Science Program model without Folders and Groups.
